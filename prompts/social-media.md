# Social Media Prompts — Pet Playground

## Platform: Facebook & Instagram

---

## Prompt 1: Before & After Post
```
Write a Facebook/Instagram caption for a before-and-after photo of a pet services job.
Tone: Proud, warm, inviting engagement.
Include: A short description of the transformation, 1-2 relevant hashtags, and a CTA (e.g. DM us for a quote).
Length: 3-5 sentences.
```

## Prompt 2: Promo / Offer Post
```
Write a social media post announcing a special offer from Pet Playground.
Offer: [describe offer here]
Tone: Exciting but not pushy.
Include urgency (limited time) and a clear CTA.
```

## Prompt 3: Trust-Builder Post
```
Write a short social post for Pet Playground that builds trust.
Use a customer review or stat (e.g. 5-star rating, years in business, jobs completed).
Keep it under 100 words.
End with a soft CTA.
```

## Prompt 4: Educational / Tips Post
```
Write a social media post sharing 3 quick tips related to pet services.
Tone: Helpful, expert, not salesy.
Audience: Homeowners or local community.
Include a question at the end to drive comments.
```
